# Getting Help — the escalation map

When the FAQs don't cover it, go in this order:

1. **Ask Claude itself.** Genuinely the best first step for how-do-I questions —
   it can search current docs. Works best in a fresh chat with a precise description.
2. **Check status.** https://status.claude.com — if the service is degraded, the
   problem isn't you. Wait it out.
3. **Official docs by product:**
   - Claude.ai / Chat / plans: https://support.claude.com
   - Claude Code: https://docs.claude.com/en/docs/claude-code/overview
   - API / developers: https://docs.claude.com
   - Product announcements: https://www.anthropic.com/news
4. **Contact support** via the help center (https://support.claude.com → contact
   options). Include: what you tried, exact error text, your platform, and a
   screenshot. Precise reports get fast answers.
5. **In-product feedback:** thumbs-down on responses in Chat; `/bug` in Claude Code.
   These genuinely reach Anthropic.

## Writing a good bug report (60 seconds, saves days)

```text
What I was trying to do:
What I expected:
What happened instead (exact error text):
Steps to reproduce:
Platform (OS, app vs web, plan):
What I already tried:
```
