# CLAUDE.md — study-timer

A tiny Pomodoro-style timer used as a teaching project. Audience: someone in their
first-ever Claude Code session.

## Commands
- Run it: `python timer.py [study_minutes] [break_minutes]`
- Quick test run: `python timer.py 0.1 0.1`

## Workflow
- Explain changes in plain English — the user may not read Python.
- Always show and explain the diff before applying a change.
- Keep this a single-file program; no new dependencies.
