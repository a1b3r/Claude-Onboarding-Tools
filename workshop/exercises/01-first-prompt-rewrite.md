# Exercise 1 — The Prompt Rewrite (pairs, 20 min)

**Goal:** experience the difference between a lazy prompt and a briefed one, on a
task you actually care about.

## Setup
Pairs. Each person needs a real task they brought (email to write, doc to improve,
topic to study, decision to make).

## Steps
1. **(3 min)** Each person writes the *lazy* version of their prompt — the one-liner
   you'd type without thinking ("fix my essay"). Run it. Save the output.
2. **(7 min)** Rewrite using the pattern — role, process, standard, guardrail — and
   add real context (paste the draft, name the audience, set the format/length).
   Run it.
3. **(5 min)** Compare outputs with your partner. Each person names the single
   change that mattered most.
4. **(5 min)** Bonus round: take the strong output and *iterate once* — one specific
   piece of feedback ("shorter", "warmer", "cut point 2"). Notice it's a
   conversation, not a slot machine.

## Debrief questions (facilitator)
- What did the strong prompt include that the lazy one assumed Claude could guess?
- Did anyone's guardrail visibly change the output?
- What's the prompt you'll reuse this week? (Point them to `prompts/` to save it.)
