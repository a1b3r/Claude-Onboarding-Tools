# Workshop Kit

> 📖 The live-session version of the [interactive guide](../guide/index.html).
> Designed so a non-expert facilitator (TA, career-center staff, club officer) can
> run it from the materials alone.

## Two formats

| Format | Time | Covers | Best for |
|---|---|---|---|
| **Intro session** | 60 min | Ecosystem map, prompting pattern, one live exercise | Orientations, club meetings |
| **Deep dive** | half day (~3.5 h) | All of the above + Cowork relay, Code demo, ethics scenarios | Bootcamps, onboarding days |

## What's in the kit

- [`facilitator-guide.md`](facilitator-guide.md) — timing blocks, talking points,
  common questions, what trips people up, setup checklist.
- [`slides/workshop-deck.md`](slides/workshop-deck.md) — the deck as Markdown
  (one slide per `---`). Present it as-is from any Markdown previewer, convert with
  [Marp](https://marp.app), or — fittingly — hand it to Claude to convert to PPTX
  or rebuild in Claude Design.
- [`exercises/`](exercises/) — four facilitated exercises with instructions,
  timings, and debrief questions.
- [`handouts/`](handouts/) — a one-page cheatsheet and an annotated CLAUDE.md
  template, designed to be printed or shared as files.

## Participant prerequisites

- A Claude account (free tier is fine for the intro session).
- A laptop. For the deep dive's Code segment: Node.js installed beforehand saves
  20 minutes — put it in the invite email (template in the facilitator guide).
