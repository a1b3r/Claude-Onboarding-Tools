# Claude AI Toolkit

A practical training toolkit for helping people use Claude Chat, Claude Code, and Claude Cowork effectively.

This repository includes beginner-friendly guides, sample prompts, Claude Code demos, Claude Cowork workflows, troubleshooting FAQs, and workshop materials designed for enterprise or university users.

## Skills Demonstrated

- Teaching Claude to users at different skill levels
- Prompt debugging and workflow troubleshooting
- Designing hands-on AI workshops
- Creating onboarding and training documentation
- Supporting responsible and privacy-aware AI usage
- Explaining technical concepts clearly to non-technical users
